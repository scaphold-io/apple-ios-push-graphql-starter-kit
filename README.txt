# Scaphold.io's Apple iOS Push Notifications Starter Kit

Fork this boilerplate code to get started with iOS push notifications.

Check out the [Scaphold Blog](https://scaphold.io/community/blog/) for guides on how to configure Scaphold to work with APNS
and start sending push notifications today!

Quickstart:

1) Go to Scaphold.io (https://scaphold.io).

2) Create an account and dataset.

3) Configure iOS Push integration.

4) Double click the .xcodeproj file to open.

5) Build the app on an actual Apple device and once the app loads, you'll see a device token in the XCode console.
